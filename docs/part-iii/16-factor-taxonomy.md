# 16 常见股票因子体系

> 所属模块：Part III 股票多因子研究

**先有风格地图，再谈「另类因子」——多数所谓新因子，只是旧风格的变装。**

## 本节导读

情景：新人提交「原创因子」：20 日涨幅除以波动率。导师问：和 Risk-Adjusted Momentum 什么关系？对 Size、Industry 暴露多少？2017 小盘股行情里是否唯一有效？——答不上来，因为缺少 **因子族（Factor Taxonomy）** 地图。

本章提供 A 股多因子常用的因子族分类：价值、质量、成长、动量、反转、波动、流动性、股东与分析师等。目标是建立共同语言，知道每类因子的 **经济直觉、典型构造、A 股制度约束与失效模式**，并在开题时先做「是否已有同类」检索。

## 学习目标

1. 识别主流因子族及其经济直觉与代表指标
2. 理解因子拥挤、风格周期与 A 股制度对各类因子的影响
3. 在开题时先做同类检索，评估增量而非重复
4. 建立团队因子词典的结构化框架

---

## 16.1 价值 Value

### 经济直觉

**用价格相对基本面的便宜程度定价。** 经典解释：风险补偿（distressed risk）、行为金融（过度悲观）、或两者混合。

### 代表因子

| 因子 | 构造 | 方向 | A 股注意 |
| --- | --- | --- | --- |
| EP | 净利润 TTM / 市值 | 高 EP 便宜 | 亏损股 EP 缺失 |
| BP | 净资产 / 市值 | 高 BP 便宜 | 金融业 BP 常系统性偏高（低 PB），全市场排序易变成行业 bet |
| SP | 营收 / 市值 | 高 SP 便宜 | 适合亏损但营收大 |
| CF/P | 经营现金流 / 市值 | 高便宜 | 现金流质量 |
| 股息率 | 分红 / 市值 | 高便宜 | 除息与复权（第 12 章） |

### 典型风险与失效

- **价值陷阱（Value Trap）**：便宜因为基本面恶化，EP 高但盈利下滑
- **周期行业**：钢铁、煤炭 EP 随 commodity 周期剧烈波动
- **与杠杆、规模纠缠**：未中性化时价值 ≈ 小盘 + 银行低配

### A 股特征

2017 前后大盘白马（龙头抱团）与小盘价值的风格切换明显；2019–2021 核心资产行情中纯价值因子阶段性失效。**价值因子必须报告行业中性化前后 IC。**

## 16.2 质量 Quality

### 经济直觉

**高质量公司应享有溢价或更抗跌**——盈利稳定、低杠杆、低应计、高 ROE。

### 代表因子

| 因子 | 构造 | 直觉 |
| --- | --- | --- |
| ROE / ROA | 盈利 / 权益或资产 | 资本效率 |
| 盈利稳定性 | ROE 时序标准差倒数 | 可预测性 |
| 低应计 | 应计利润 / 总资产 | 盈余质量 |
| 低杠杆 | 负债 / 权益 | 财务风险 |
| Gross Profitability | 毛利润 / 资产 | Novy-Marx 经典 |

### 典型风险

- 会计操纵：应计、商誉、非经常性损益
- **PIT 对齐**：财报可得日滞后（第 13 章）
- 与 **成长、动量** 正相关：「好公司」往往已涨价

### A 股特征

注册制后退市与财务造假处罚趋严，质量因子对 **ST 预警** 有一定区分力；但需注意银行业 ROE 口径与制造业不可比——**行业内标准化** 常见。

---

## 16.3 成长 Growth

### 经济直觉

**盈利或收入的增长速度** 应预测未来超额——增长尚未 fully priced in。

### 代表因子

| 因子 | 构造 | 注意 |
| --- | --- | --- |
| 净利润 YoY | $(\mathrm{NI}_{t}-\mathrm{NI}_{t-4})/\lvert\mathrm{NI}_{t-4}\rvert$ | 基数效应 |
| 营收 YoY | 同上 | 亏损公司仍可用 |
| SUE | $(\mathrm{Actual}-\mathrm{Expected})/\sigma$ | 需一致预期数据 |
| 长期增长预期 | 分析师一致预期增长 | 覆盖度偏大盘 |

### 与动量、估值的交织

高增长股票往往 **高估值 + 高动量**；单独成长因子 IC 可能实为 **动量或 Size 马甲**。增量检验：对 EP、MOM 回归取残差（第 20 章）。

### 失效模式

- 宏观下行期：增长溢价消失，「戴维斯双杀」
- 主题炒作：短期增长叙事与长期 fundamentals 脱节

---

## 16.4 动量 Momentum

### 经济直觉

**中期过去收益延续**——行为惯性、机构调仓滞后、盈利趋势持续。

### 代表因子

| 因子 | 构造 | A 股参数 |
| --- | --- | --- |
| MOM_20 / MOM_60 | 过去 N 日累计收益（后复权） | N=20/60/120 常见 |
| 52 周高点距离 | close / max(close, 252d) - 1 | 接近 0 为强势 |
| 行业相对动量 | 个股收益 - 行业收益 | 降行业 beta |

### A 股制度约束

- **涨跌停**：强势动量股买不进（第 12 章）
- **停牌**：收益序列断裂
- **T+1**：信号 T 日，最早 T+1 成交
- **高换手**：动量换仓快，成本敏感

### 失效与拥挤

2015 小盘股动量极端；2021 后量化私募容量扩张，动量 **拥挤度** 上升，回撤对称且快。需监控 IC 衰减与因子收益相关性（第 19 章）。

---

## 16.5 反转 Reversal

### 经济直觉

| 持有期 / horizon | 机制 | 代表 |
| --- | --- | --- |
| 短期（1–5 日） | 微观结构、过度反应修正 | 短期反转 |
| 长期（3–5 年） | 均值回归、估值锚 | 长期反转（A 股研究较少） |

### 代表因子

- **REV_5d**：过去 5 日收益取负
- **隔夜 vs 日内**：A 股散户占比高，隔夜与日内收益分解有文献支持

### A 股注意

- 涨跌停导致 **虚假反转**：跌停后无法继续跌，统计上像「反弹」
- 与 **流动性、Size** 强相关：小票短期反转更显著
- 成本极高：日频换仓不现实，常作 **辅助因子或约束** 而非主 Alpha

---

## 16.6 波动与残差波动

### 经济直觉

**低波动异象（Low Volatility Anomaly）**：高风险并未获得成比例补偿，低波股票风险调整后更优。

### 代表因子

| 因子 | 构造 | 说明 |
| --- | --- | --- |
| 历史波动率 | std(ret, 60d) | 低波为优 |
| 特质波动率 IVOL | 对 Fama-French 回归残差 std | Ang et al. |
| Beta | 对指数回归 slope | 低 beta 异象 |
| 最大日跌幅 | min(ret, 60d) | 尾部风险 |

### 中性化必要性

波动与 **市值、行业、杠杆** 高度相关——未中性化的「低波」常是 **大盘 + 公用事业**。标准流程：行业 + 市值中性化后再检验 IC（第 17 章）。

---

## 16.7 流动性与换手

### 经济直觉

| 方向 | 理论 | 因子 |
| --- | --- | --- |
| 非流动性溢价 | 流动性差需补偿 | Amihud Illiq |
| 高换手惩罚 | 过度交易、拥挤 | 20 日平均换手率 |

### 代表构造

$$\text{Amihud}_t = \frac{1}{D}\sum_{d=1}^{D}\frac{|r_d|}{\text{Amount}_d}$$

- **Turnover**：过去 20 日平均换手率——高换手在 A 股常 **负向** 预测收益
- **Amount / Cap**：成交额相对市值——流动性深度

### 容量研究必看

多因子产品规模上亿后，**流动性因子约束** 决定可承载 AUM。小盘高 Illiq 因子 IC 高但 **容量极低**——须在 Spec 中标注 `capacity_tier`。

## 16.8 其他常见类别

### 股东与资金

| 类别 | 示例 | 数据特点 |
| --- | --- | --- |
| 北向资金 | 持股变化、占流通比 | 日频披露，滞后 |
| 融资融券 | 融资余额变化 | 偏大盘 |
| 股东增减持 | 高管、大股东 | 事件驱动，PIT |
| 机构持仓 | 基金季报重仓 | 季频，严重滞后 |

### 分析师

- **一致预期修订**：EPS forecast 上调
- **覆盖度**：分析师数量 proxy 关注度
- 覆盖偏 **大中盘**，小票缺失严重

### 情绪与事件

- 新闻 NLP 情绪、龙虎榜、限售解禁
- 更依赖另类数据与 **严格 PIT**（第 09、13 章）

### 拥挤度（Meta Factor）

- 因子收益与同类产品持仓相关
- 用于 **风控降权**，非传统 Alpha

### 因子族相关矩阵（经验认知）

```text
        Value  Quality  Growth  MOM   Rev   Vol   Liq
Value     1      +        -      -     +     +     +
Quality   +      1        +      +     -     -     -
Growth    -      +        1      +     -     -     -
MOM       -      +        +      1     -     -     -
```

（符号为典型方向，具体需样本估计——**不可背诵，要算**）

### 因子族与产品目标

| 产品 | 更关注 |
| --- | --- |
| 指数增强 | 在行业/风格约束下的增量选股 |
| 市场中性 | 对冲后残差收益，更怕风格残留 |
| 量化多头 | 绝对收益与回撤，风格择时可能并存 |

同一价值因子，在不同产品里的「是否保留原始暴露」答案可以不同。

---

## 常见错误

- 把高度相关的因子（EP、BP、SP）当「三个独立 Alpha」——有效自由度远小于 3
- 忽视风格本身的周期性失效——2017 价值、2021 动量各有大幅衰减阶段
- 不查已有因子库就重复造轮子——团队内 EP 已有五个变体
- 忽视 A 股制度：做动量却不处理涨跌停可成交性；做成长期却不按公告日 / PIT 对齐
- 把行业暴露当 Alpha 宣称——未中性化就入库
- 用故事包装数据挖掘出的奇怪比值

## 要点回顾

- 因子族提供共享语言：价值、质量、成长、动量、反转、波动、流动性、资金/分析师
- 每类有经济直觉、代表构造、A 股制度约束与典型失效模式
- **增量贡献** 比「名字新」更重要；先地图，再挖井
- 风格周期性是常态，入库因子需跨周期稳健性（第 19 章）
- 产品目标决定是否保留风格暴露
