# Part III 股票多因子研究

**因子不是魔法数字——它是可检验的经济假设在数据上的投影。**

## 本节导读

Part I 建立了「量化研究是什么」，Part II 解决了「数据从哪里来、怎么对齐」。Part III 进入 handbook 的 **核心模块**：如何把经济直觉变成可计算的因子，如何检验它是否真有用，以及如何把它与其他因子组合而不掉进过拟合陷阱。

本 Part 面向 **A 股中低频股票多因子** 场景——指数增强、量化多头、股票市场中性。所有方法都强调三件事：**经济逻辑**（为什么有效）、**可交易性**（扣成本后还能不能做）、**稳健性**（换样本、换参数是否仍成立）。

## 本 Part 的统一研究范式

每研究一个因子，建议按以下证据链推进——这也是各章内容的隐含主线：

```mermaid
flowchart LR
    A[经济假设] --> B[因子定义]
    B --> C[数据处理]
    C --> D[统计检验]
    D --> E[组合收益]
    E --> F[风险归因]
    F --> G[样本外与稳健性]
```

**不要跳步。** 许多新人直接跳到 IC 和分组回测，却说不清「为什么 PE 低应该涨」——这种因子在 A 股风格切换时最先失效。

## A 股多因子研究的三个硬约束

| 约束 | 含义 | 典型踩坑 |
| --- | --- | --- |
| 可交易性 | 涨跌停、停牌、ST、流动性 | 回测满仓小票，实盘买不到 |
| 数据可得性 | 财报公告滞后、PIT 对齐 | 用报告期而非公告日 → 未来函数 |
| 容量与成本 | 换手、冲击、印花税 | IC 0.05 但日频调仓 → 成本吃光 Alpha |

- Part III 是 handbook 核心：从认知 → 体系 → 构造 → 检验 → 稳健 → 组合。

- 18 章单因子检验是实操枢纽；19、20 章决定因子能否进生产组合。
