# 19 因子稳健性与有效性

> 所属模块：Part III 股票多因子研究

**样本内好看只是入场券；稳健性才是能不能进组合的门票。**

## 本节导读

情景：某反转因子全样本 Rank IC=0.06、ICIR=1.2，研究员申请入库。稳健性组一切子样本：2015 年 IC=0.15（杠杆牛市），2021 年 IC≈0，2022 年 IC 为负；参数从 5 日改 7 日 IC 大幅衰减；扣 30bp 成本后多空 Sharpe 从 1.5 跌至 0.3——**单因子检验（第 18 章）通过，不等于可以上线**。

单因子检验通过后，还要问：换一段市场行不行？换一套参数行不行？换一个宇宙行不行？成本后还在不在？本章给出 **稳健性（Robustness）清单**，帮助识别过拟合与伪发现，作为因子入库的硬性门槛。

## 学习目标

1. 设计子样本与制度切换检验，识别「单段行情因子」
2. 实施参数敏感性与 IC 衰减分析
3. 把交易成本、换手、风格暴露纳入「有效性」定义
4. 建立模板化稳健性报告与季度再评估机制

## 19.1 子样本稳健性

### 为何要切子样本

全样本 IC 是 **单一数字**，掩盖结构性失效：因子可能只在 2015 小盘疯牛、或 2020 核心资产行情中有效，换环境即归零。

### 时间维度切分

| 切法 | 目的 | A 股参考段 |
| --- | --- | --- |
| 滚动窗口 | IC 时序稳定性 | 滚动 24 个月 Mean IC（亦可用 12/36；窗口须事先约定） |
| 牛/熊/震荡 | 市场状态依赖 | 2014–15 牛、2018 熊、2019–21 结构 |
| 制度切换 | 政策冲击 | 2016 熔断、2019 注册制、2021 量化监管 |
| 样本外 OOS | 最终检验 | 最后 20% 时间 hold-out |

```python
import pandas as pd

def subsample_ic(ic_series: pd.Series, periods: dict) -> pd.DataFrame:
    """
    ic_series: index=trade_date, values=daily IC
    periods: {"2015_bull": ("2014-10-01", "2015-06-30"), ...}
    """
    rows = []
    for name, (start, end) in periods.items():
        sub = ic_series.loc[start:end]
        rows.append({
            "period": name,
            "mean_ic": sub.mean(),
            "icir": sub.mean() / sub.std() if sub.std() > 0 else float("nan"),
            "n_days": len(sub),
        })
    return pd.DataFrame(rows)

def ic_by_year(ic: pd.Series) -> pd.DataFrame:
    return ic.groupby(ic.index.year).agg(["mean", "std", "count"])
```

### 截面维度切分

| 切法 | 问题 |
| --- | --- |
| 大/中/小盘 | 是否只是小票效应 |
| 行业 | 是否只在 TMT、医药有效 |
| 指数内外 | 沪深 300 vs 中证 1000 |
| 可交易子样本 | 剔除涨停买不进后 IC 是否还在 |

### 判定标准（经验）

- 子样本 Mean IC **符号一致** 且 **|IC| > 全样本 50%** 的段数 ≥ 总段数 70%
- 任一段 |IC| > 2× 全样本且其余段≈0 → **单段行情因子**，降权或否定
- OOS 段 IC 与 IS 段同号且 t 值 > 1.5（月频约 31 点）为底线

### 情景：行业集中因子

某股东增减持因子 IC 全样本 0.05，切行业后发现 **80% IC 来自房地产**——2016 棚改周期特有，2022 后失效。结论：不可作为 broad Alpha，最多作事件 overlay。

## 19.2 参数敏感性

### 参数网格

对 winsor 分位、回看窗口、滞后期、中性化是否开启等做 **预注册网格**：

| 参数 | 典型网格 | 红旗 |
| --- | --- | --- |
| 动量窗口 | 10, 20, 40, 60, 120 日 | 仅 23 日有效 |
| winsor | 1%, 5%, MAD k=3,5 | 尖峰分布 |
| 持有期 | 1, 5, 10, 20 日 | 与窗口共线 |

### 参数曲面

报告 **Mean IC 热力图**（x=窗口, y=持有期），而非只报最优点。健康因子应在 **平坦区域** 内为正，而非孤立尖峰。

| 健康形态 | 危险形态 |
| --- | --- |
| 一片参数区域同号 | 只有一个尖峰参数赚钱 |
| 邻近参数平滑 | 改 1 天窗口就变号 |

```python
def param_grid_ic(panel_fn, param_grid: list) -> pd.DataFrame:
    """
    panel_fn(params) -> ic_series
    param_grid: [{"window": 20, "hold": 5}, ...]
    """
    results = []
    for p in param_grid:
        ic = panel_fn(**p)
        results.append({**p, "mean_ic": ic.mean(), "icir": ic.mean() / ic.std()})
    return pd.DataFrame(results)
```

### 预注册 vs p-hacking

**禁止**：试 50 组参数只报 IC 最高的一组。**正确**：开题时写 Parameter Grid，全部跑完附在附录；主结论用 **网格中位数** 或 **默认经济含义参数**（如 20 日动量）。

### 与 17 章构造的关系

winsor、中性化参数应在 Spec **固定**；19.2 检验的是 **经济参数**（窗口、持有期），不是事后调 winsor 救 IC。

## 19.3 衰减与拥挤

### IC 衰减

观察 Mean IC、ICIR 的 **滚动 24 月曲线** 是否趋势下滑：

- 缓慢下滑：可能拥挤或逻辑弱化
- 断崖下跌：常伴随 **同类因子资金涌入** 或 **监管/制度变化**

### 因子收益衰减

除 IC 外，跟踪 **多空组合累计收益** 的斜率变化——IC 仍正但 LS flatten，可能是 **成本上升或执行摩擦**。

### 拥挤信号

| 指标 | 含义 |
| --- | --- |
| 因子换手上升 | 更多资金追逐同一信号 |
| 与知名产品持仓相关上升 | 拥挤 |
| 同类因子 IC 同步下滑 | 集体失效 |
| 小票容量因子 AUM 扩张 | 容量瓶颈 |

### 情景：2021 量化回撤

多家指增同配 **500 增强 + 动量 + 价值**，2021 年 9 月后同步回撤——单因子 IC 仍正但 **因子收益相关=0.8**，说明拥挤而非单因子 bug。

### 停用触发器（建议写入 Spec）

| 触发条件 | 动作 |
| --- | --- |
| 滚动 12 月 Mean IC 变号 | 降权 50% |
| 滚动 12 月 ICIR < 0.3 | 暂停入库合成 |
| 与 cluster 代表相关 > 0.9 持续 6 月 | 冗余审查 |

## 19.4 成本与可交易性

### 有效性定义

**有效性 = 统计显著 + 经济显著 + 成本后可实现**

| 成本项 | A 股典型（2024 参考，示意） |
| --- | --- |
| 佣金 | **单边**约 0.03%（万三）；双边约 0.06% |
| 印花税 | 卖出约 0.05%（政策可调） |
| 滑点 | 单边约 5–30bp，随市值变化 |
| 冲击 | 大单额外，Part V 展开 |

### 简易成本模型

```python
def apply_trading_costs(
    gross_ret,
    turnover,
    commission_1way: float = 0.0003,
    stamp: float = 0.0005,
    slippage_1way: float = 0.001,
):
    """
    gross_ret: 组合毛收益序列
    turnover: 单边换手（买入权重之和，或 0.5×双边换手）
    参数均为单边费率；印花税仅卖出，近似用 turnover × stamp
    """
    cost = turnover * (2 * commission_1way + stamp + 2 * slippage_1way)
    return gross_ret - cost
```

### 可交易性叠加

- 涨停买入失败率（第 12 章）
- 停牌无法调仓
- 最小成交额过滤

**报告三轨**：毛收益、扣费收益、可成交扣费收益——逐级严苛，产品看第三轨。

### 情景：高 IC 低 Sharpe

IC=0.05 但因子自相关仅 0.3（周频几乎换仓），年化换手 800%，印花税+滑点吃掉全部 alpha——**ICIR 高不等于 IR 高**。

## 19.5 风险调整后的有效性

### 增量 Alpha 检验

对 **Fama-French 风格 + 行业** 回归，检验因子收益是否仍有显著 $\alpha$：

$$r_{i,t+1} = \alpha + \beta_m MKT + \beta_s SMB + \beta_v HML + \cdots + \beta_f f_{i,t} + \varepsilon$$

或在 **因子组合层面**：用已有 Barra 风格因子对目标因子 LS 收益回归，看残差 Sharpe。

### 暴露诊断

| 检验 | 通过标准（经验） |
| --- | --- |
| 行业暴露 | 中性化后 \|t\| < 2 |
| 市值暴露 | \|beta_ln_cap\| < 0.1 |
| 对 EP/MOM 增量 | 正交后 IC > 0 且 t > 2 |

若正交后 IC≈0：因子是 **风格马甲**，仍有配置价值，但不应宣称「纯 Alpha」。

### Fama-MacBeth 与 IC 互补

第 18 章 Fama-MacBeth 给出 **控制变量后的 risk premium**；19.5 强调：**控制后 $\lambda$ 仍显著** 才是入库级 Alpha。

对指数增强，还应看：**在跟踪误差预算内是否提高 IR**。

## 常见错误

- 只报最好子样本（2015 段）——cherry-picking
- 参数搜完再讲故事——p-hacking
- 忽略成本谈稳健——实盘 IR 为负
- 把 OOS 当「再试一次 IS」——OOS 只能用一次做主结论
- IC 正但 LS 单调性崩——未交叉验证
- 拥挤期仍满配——未做 meta 降权
- 没有停用触发器就上线——失效时无法决策

## 要点回顾

- 稳健性 = 跨时段 / 宇宙 / 参数 / 成本 / 增量暴露仍成立
- 参数岛与单段行情是 **红旗**；报告网格而非最优点
- IC 衰减与拥挤需持续监控；设定停用触发器
- 有效性必须 **成本后、可成交后** 仍显著
- 风险调整后谈增量 Alpha 更诚实
