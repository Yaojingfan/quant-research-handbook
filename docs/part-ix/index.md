# Part IX 从研究到实盘

**回测通过不是终点；能日复一日稳定跑完生产闭环，才算策略真正「活着」。**

本 Part 覆盖模拟盘验证、每日生产流程、监控故障处理与容量管理——把研究结论接到真实交易约束上。

## 本 Part 的目标

1. 用 Paper Trading / Shadow 对齐回测与实盘偏差
2. 理解日频生产链路：数据 → 因子 → 组合 → 风控 → 下单 → 归因
3. 设计异常分级与降级、回滚、停机机制
4. 评估容量、冲击与规模扩大后的收益衰减
