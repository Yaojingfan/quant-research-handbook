# Quant Research Handbook

# 量化策略研究新人手册

> **收益来自可重复的过程，而不是某一次漂亮的回测曲线。**

面向 A 股股票多因子策略团队的内部研究手册。目标不是堆叠文章，而是帮助新人建立**可执行、可复现、可评审**的量化研究工作方式——从数据到因子，从组合到回测，从风险归因到生产部署。

---

## 这本手册讲什么？

不是策略圣杯，而是教你**如何做量化研究**。市面上的教程常停留在 API 翻译、指标堆砌、过拟合的"神奇因子"。真正的研究体系需要回答：

| 问题 | 对应模块 |
| --- | --- |
| 数据从哪来？口径是否一致？有没有未来函数？ | Part II 数据基础 |
| 因子为什么可能有效？如何检验而非"调参出 Alpha"？ | Part III 因子研究 |
| 信号如何变成可约束、可交易的投资组合？ | Part IV 组合构建 |
| 回测怎么不骗自己？成本与偏差如何处理？ | Part V 回测体系 |
| 收益来自哪里？风险暴露是否失控？ | Part VI 风险与归因 |
| 研究如何工程化、可复现？ | Part VII 研究工程化 |
| 机器学习如何定位（可选延伸）？ | Part VIII 机器学习 |
| 如何接到模拟盘与生产？ | Part IX 从研究到实盘 |

```mermaid
flowchart LR
    A[数据] --> B[因子]
    B --> C[组合]
    C --> D[回测]
    D --> E[风险与归因]
    E --> F[工程化]
    F --> G[实盘]
```

**研究边界**：本手册以**股票中低频多因子**为核心，重点覆盖指数增强与股票市场中性；CTA、统计套利作简介；高频、做市、数字资产仅概念性提及。
