# Part VII 研究工程化

**研究不可复现，就等于没有研究——只有一次性表演。**

本 Part 把「会算因子」升级为「可协作、可审计、可回滚」的研究工程：Python/SQL 习惯、代码结构、实验记录、研究平台边界。

## 本 Part 的目标

1. 掌握量化研究常用的 Pandas / SQL 模式
2. 用模块化与配置驱动减少一次性脚本
3. 绑定代码版本、数据版本与实验结果
4. 理解研究平台应提供的最小能力集
