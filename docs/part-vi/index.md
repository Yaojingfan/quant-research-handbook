# Part VI 风险管理与收益归因

**不会归因的研究员，很难知道自己赚的是 Alpha 还是风格运气。**

回测曲线向上，不等于选股能力强。Part VI 教你用风险框架拆解收益：市场、行业、风格、特质收益各贡献多少；何时该降仓、何时该怀疑因子失效。

## 本 Part 的目标

1. 建立事前风险与事后风险的基本框架
2. 理解多因子风险模型的用途与局限
3. 能做基础的 Brinson / 因子归因解读
4. 设计监控指标识别策略衰减与异常暴露
