# Part IV 从因子到投资组合

> **因子显著不等于能赚钱。** 从横截面排序到可交易组合，中间隔着股票池、预期收益映射、权重分配、风险约束与优化器——每一步都在消耗 Alpha，也都在暴露估计误差。

## 本节导读

Part III 回答了「因子有没有效」；Part IV 回答「如何把因子变成可持有的组合」。本 Part 覆盖从 **股票池定义** 到 **组合优化** 的完整链路，核心关切是：**在 A 股涨跌停、停牌、容量与基准约束下，你的信号还能剩多少？**

## 学习目标

1. 构建无幸存者偏差、可交易的研究股票池
2. 将因子分数映射为预期收益与目标权重
3. 理解风险约束、中性化与优化器的适用边界
4. 在「简单规则」与「复杂优化」之间做出有依据的选择
